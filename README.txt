Создать небольшую базу данных для магазина автомобилей, на примере Toyota.

База формируется на основе взаимосвязей сущностей
- Модель -> Доступные комплектации модели -> Доступные цвета для модели

Yaris Elegant Red
Yaris Elegant Black
Yaris Active Red
....

Model
-- List < Modification>

Modification
-- List <Color>

Каждая сущность содержит:
ID - GUID - уникальный индификатор в базе даннх
Name - String - название для отображения
VendorId - String - Код по таблице производителя

Например:
VendorId Model Modification Color
NSP131L-CHXNBW Yaris Active 040 - Белый Лак

Реализовать
CRUD для данных
-- Создание модели, создание модификаций в модели, создание цветов для данной модификации
-- Изменение и удаление для сущностей на любом уровне
-- Сохранение в XML файл

Search (Filter)
-- Вывести все машины, доступные в желаемом цвете

+ будет хранение доступных цветов в отдельной клекции
